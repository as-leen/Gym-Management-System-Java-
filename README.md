# Gym Management System

## Overview
The Gym Management System is a Java-based application designed to manage gym members, items, and dietary supplements.  
The project focuses on organizing gym operations using object-oriented programming concepts.

It demonstrates structured class design and data handling in Java.

## Features
- Gym member management
- Item and dietary supplement tracking
- Date handling for records
- Organized class-based structure
- Console-based interaction

## Technologies Used
- Java
- NetBeans IDE
- Object-Oriented Programming (OOP)

## Project Structure
src/  
├── Main.java  
├── NewMain.java  
├── Member.java  
├── Item.java  
├── DietarySupplement.java  
├── Date.java  
└── basic.java  

## How to Run
1. Open the project using NetBeans IDE  
2. Clean and build the project  
3. Run Main.java or NewMain.java  
4. Interact with the system through the console  

## Learning Outcomes
- Understanding OOP fundamentals
- Managing relationships between classes
- Designing structured Java applications
